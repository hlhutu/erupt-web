## ⛰ 环境搭建

在开始之前，请确保你的开发环境中包括 Node.js 和 npm 包管理器。

Angular 需要 Node.js 版本 10.9.0 或更高版本。

要检查你的版本，请在终端/控制台窗口中运行 node -v

## 🥌 使用方法

1. 安装 yarn
```shell script
npm install -g yarn
```

2. 构建：
```shell script
yarn install
```

3. 修改服务端地址
```
/proxy.config.js → target
```

4. 启动：
```shell script
yarn run start
```


## 🍭 打包编译：
1. 修改 angular.json 打包路径：
> projects.architect.build.options.outputPath

2. 执行打包命令
```shell script
yarn run build
```
